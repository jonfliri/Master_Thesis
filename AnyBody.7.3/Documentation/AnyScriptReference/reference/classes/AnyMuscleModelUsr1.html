AnyMuscleModelUsr1¶

Type

Creatable

Parent class

AnyMuscleModel

Description¶

This class gives the possibility to make simple user-defined muscle models. The strength of the muscle can be defined by an explicit expression for the member ‘S’. This expression is evaluated and used in the muscle recruitment analysis during for instance inverse dynamic simulation.

The strength ‘S’ can be a function of time, position and velocity variable, so that force-length and force-velocity relationships can be implemented. Notice, that any dependency between the strength and other muscle model parameters must be made explicitly. This suggests to make ‘S’ and function of ‘F0’, which then again could be a function of ‘Vol0’ and ‘Lf0’. Such explicit dependence in the expression for ‘S’ enable that ‘S’ follows volume changes e.g. during model scaling or optimal fiber length changes e.g. during calibration.

Notice that this muscle model does not hold passive muscle properties. Using this model, passive properties must be implemented as a separate force object.
