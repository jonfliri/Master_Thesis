AnyChartCursor¶

Type

Abstract

Parent class

AnyChartObject

Description¶

Chart cursor object. A cursor is attached to a series. It work for both curve and surface plots. For curve plots the cursor use the first element of the members such as AbsIndex, etc., whereas it uses both elements for surface plots.

Notice that the cursor position typically can be modified by dragging it in the Model View or the AnyChart View.
