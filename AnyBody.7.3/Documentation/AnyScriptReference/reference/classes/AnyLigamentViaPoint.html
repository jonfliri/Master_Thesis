AnyLigamentViaPoint¶

Type

Creatable

Parent class

AnyLigament

Child classes

AnyViaPointLigament

Description¶

This is a specilization of the general ligament class, AnyLigament, that connects two or more so-called via points. The via points are specified as reference frames put into the object as members.

Notice that this ligaments kinematics assumes the muscle to be a string that can wrap over edges, but it will also stick to such edges, so that the normal forces can become negative.
