AnySceneObject¶

Type

Abstract

Parent class

AnyModelViewObject

Child classes

AnyScene, AnySceneForceDrawing, AnySceneViewer

Description¶

This is the base class for all object that in some way define the model scene.

This includes object containing scene settings (AnyScene), light objects (AnyLight), …
