AnyMuscleModel2ELin¶

Type

Creatable

Parent class

AnyMuscleModel

Description¶

This class contains a two element, Hill-type muscle model. The two elements are the contractile element (the muscle fibers) and the serial elastic element (the tendon). The contractile element is modeled by a bi-linear function, i.e., the force-length and the force-velocity relationships are linear. The elastic tendon force is also modeled by a linear elastic relationsship.

Input parameters, ‘Lfbar’ and ‘Epsilonbar’ are deprecated members, which were used before AnyBody version 7.1. Now they are replaced by similar values where ‘bar’ is replaced by ‘0’. New and deprecacted values have the same meaning.
