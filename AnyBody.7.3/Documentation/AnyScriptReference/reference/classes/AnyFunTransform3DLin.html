AnyFunTransform3DLin¶

Type

Creatable

Parent class

AnyFunTransform3D

Description¶

Linear 3D transformation function.

The linear scaling is defined by a 3 by 3 scaling matrix that is multiplied to the argument (3D geometrical vector) after adding a set of offset values.

The scaling matrix (ScaleMat) and the offset vector (Offset) are given as members of the class.

Examples¶

// A linear scaling (3D transformation) function
AnyFunTransform3DLin ScaleFun = {
  AnyVec3 ScaleXYZ = {1,2,0.1};  //< Assisting vector holding size scaling factors
  ScaleMat = {{ScaleXYZ[0],0,0},{0,ScaleXYZ[1],0},{0,0,ScaleXYZ[2]}}; //< Scaling matrix with pure size scaling entries
  Offset = {0,0,0};  //< Translation part
};

Then you can apply this transformation to a given vector:

AnyVec3 ScaledVector = ScaleFun(.OriginalVector); ///<Scaled vector produced by calling the scaling function
