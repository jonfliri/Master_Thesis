AnyFunTransform3DRBF¶

Type

Creatable

Parent class

AnyFunTransform3D

Child classes

AnyFunTransform3DRBFdyn

Description¶

3D transformation function based on Radial Basis Functions (RBF).

A radial basis function (RBF) is a function, where its value depends only on the distance from the origin. In this class, a number of radial basis functions are used to define a spatial vector function, i.e., a function that provide a 3D vector as function of a 3D vector. This implies that three spatial RBF interpolations are used, each defining one of the three elements of the vector function.

Each RBF in the functions is defined by a control point, the origin for that function, and three coefficients, one for each spatial dimension. Any number of control points are given in the (n by 3) matrix Points0, where each row is a control point. The associated coefficients are similarly given in RBFCoefs. The type of RBF and its basic parameters are set in the folder RBFDef.

In addition to the contribution from each RBF, a polynomial of optional degree is added to the return value. The degree of this polynomial is given by PolynomDegree and its coefficients are given by PolynomCoefs.

Specification of the RBF interpolation by means of Points0, RBFCoefs, and PolynomCoefs is not very intuitive. This direct form of specification calls for an external tool of calculating the two sets of coefficients. Therefore, an implicit specification is available as an intuitive alternative. The matrix Points1 are target points, i.e., a 3D point associated with each control point, which implies that Points1 must have the same form as Points0. The final interpolation becomes an interpolation (and extrapolation) of the target point values over the space of the control points. The coefficients for the RBF and the polynomia are computed so that the interpolation fits at the control points.

Examples¶

AnyFunTransform3DRBF RBFTransform = {
  PreTransforms = {&.AffineTransform}; // See AnyFunTransform3DLin2 example
  RBFDef.Type = RBF_Triharmonic;
  PolynomDegree = 1;
  Points0 = .AffineTransform.Points0;
  Points1 = .AffineTransform.Points1;
  BoundingBoxOnOff = On;
  BoundingBox.Type = BB_Cartesian;
  BoundingBox.ScaleXYZ={1,2,2}*1.2;
  BoundingBox.DivisionFactorXYZ ={1,1,1}*3;
};

Coefficients for the polynomia function added to the radial basis function. Each term of the polynomia must be given three parameters, one for each direction in space, i.e., the PolynomCoefs are given as a (m x 3) matrix. The number of rows, m, must match PolynomDegree; degree zero means m=1 (the constant term), positive degrees mean m=1+3*degree, and negative degrees mean m=0.
