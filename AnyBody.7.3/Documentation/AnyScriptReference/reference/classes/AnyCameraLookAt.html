AnyCameraLookAt¶

Type

Creatable

Parent class

AnyCamera

Description¶

The LookAt camera is supposed to be an intuitively easier camera to position.

The view point is defined by three points, the position of the eye, the position of the point being looked at (the look-at-point) and a point defining what is up (the up-point). This defines the view point reference system with the z-axis parallel to the vector between the look-at-point and the eye point. The y-axis is the normal vector from the view line (z-axis) to the up-point, and the x-axis is given so a right-handed view point system is obtained.

If the camera is attached to another 3D object then the coordinates of these points are assumed to refer to object reference frame; otherwise they are global points.
