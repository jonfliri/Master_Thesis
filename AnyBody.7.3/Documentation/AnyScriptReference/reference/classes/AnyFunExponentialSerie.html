AnyFunExponentialSerie¶

Type

Creatable

Parent class

AnyParamFun

Description¶

Exponential series function.

The exponential series function calculates each element of the output according to this expression:

\[f_i(t) ~=~ C0_i ~+~ \sum_{j=1}^{n}C1_{ij}\,\cdot\,{Base}^{(C2_{ij}\,t)}\]

where \(C0_{i}, C1_{ij}, C2_{ij}, and Base\) are the elements of the members of the same name, \(i\) is the index of the return value, and \(j\) is the counter over the elements associated with the i’th return value.

The sizes and dimensionality of the return from this function is given by the member C1, which therefore must be initialized. C1 is interpreted as an array of blocks with the same dimensionality and size as the return of the function, i.e., C1 must have dimensionality one higher that the wanted return value. C2 is the exponents and must be initialized of the same sizes as C1. C0, the constant term, is optional, but must have the sizes of the return value if initialized.

In other word, if we indicate return values data block size like this \(\{XXX\}\) the member initialization must have the following form.

\(C0 ~=~ \{XXX\};\) \(C1 ~=~ \{~\{XXX\},~\{XXX\},~...~\};\) \(C2 ~=~ \{~\{XXX\},~\{XXX\},~...~\};\)
