AnyKinMeasureLinComb¶

Type

Creatable

Parent class

AnyKinMeasureCombBase

Child classes

AnyKinMeasureQuadComb

Description¶

This class is a composer of new kinematic measures as a linear combination of existing ones.

To some extent, it is similar to AnyKinMeasureOrg, but the way it composes the new measure is based on linear algebra.

The linear combination is defined by a coefficient matrix, named Coef, and constant terms in the vector named Const. Each row in Coef and Const defines one component of the new kinematic measure. Each column of Coef contains the coefficients associated with one component of the existing measures that have been entered into this new measure (either directly or typically by references).

(1)¶\[\underline{y} = \underline{\underline{A}} \underline{x} + \underline{b}\]

where \(\underline{\underline{A}}\) is the coefficient matrix (i.e. Coef), and \(\underline{b}\) is the vector of constants (i.e. Const)

All the components of the entered measures (also referred to as sub-measures) are assembled into one vector, called \(\underline{x}\) in the above expression, and \(\underline{y}\) is the vector of output measures. The assembly of inputs in \(\underline{x}\) is similar to class AnyKinMeasureOrg. The firstly defined sub-measure occupies the first places of the vector and so on. Please refer to AnyKinMeasureOrg for details of organizing/re-organizing kinematic measures.

The integer OutDim is be used to specify the number of outputs (i.e. the length of \(\underline{y}\)), and all other specifications such as the dimensions of Coef and Const must match this.

Time-varying coefficients¶

AnyKinMeasureLinComb contains options to add explicit time-varying terms to the coefficient matrix (\(\underline{\underline{A}}(t)\)) and vector of constants (\(\underline{b}(t)\)). These time-varying terms are given by the optional members CoefTimeFun and ConstTimeFun for \(\underline{\underline{A}}^+(t)\) and \(\underline{b}^+(t)\), respectively. They are both pointers to parameters-function-objects of class AnyParamFun (or derived). The function that are referred to this way will be considered as time-function, and their output must match the dimensions of members Coef and Const, respectively. It must be noticed that these time varying terms, given by CoefTimeFun and ConstTimeFun, and the constant terms in Coef and Const are added before the linear combination is calculated (Equation :eq:’eq_AnyKinMeasureLinComb’).

Please also notice that the linear combination is extended with quadratic terms in the derived class AnyKinMeasureQuadComb.

Example¶

If we for instance assemble (include) two measures with respectively 2 and 3 components, the assembled measure will have 5 components, with indices from 0 to 4. We can then define:

AnyKinMeasureLinComb ExampleMeasure = {
    AnyKinMeasure &ref1 = ..TwoComponentMeasure;
    AnyKinMeasure &ref2 = ..ThreeComponentMeasure;
    OutDim = 2;
    Coef = {
        {1, -1, 0, 0, 0},
        {0, 0, 1, 2, 3}
    };
    Const = {10, 20};

This yields a new measure with two components: the former is the difference between the components of the first sub-measure and the latter the sum of the components in the latter sub-measure.

\[\begin{split}y[0] &= x[0] - x[1] + 10 \\ y[1] &= x[2] + 2 x[3] + 3 x[4] + 20\end{split}\]

For an example of time-varying coefficients, please see the enclosed example files.
