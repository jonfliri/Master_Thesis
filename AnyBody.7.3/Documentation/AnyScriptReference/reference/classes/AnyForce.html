AnyForce¶

Type

Creatable

Parent class

AnyAppliedForce

Description¶

This class function to create a generalized applied load. It is a generalized force in the sense that it can be applied to any kinematic measure or even an assembly of such. Therefore, it is a vector quantity The term “applied load” refers to that the force vector must be an explicit function of the model state (position and velocity) and time.

Refer to the parent class AnyForceBase and its parent AnyKinMeasureUsr for further information on how the force is defined (units) and how several kinematic measures are assembled.
