AnyOprSetMuscleRecruitment¶

Type

Abstract

Parent class

AnyOprSetMechSolver

Description¶

Class specifying the optimization criterion for a muscle recruitment problem, typically solved within an AnyBodyStudy by operations of class AnyMechOprInvMuscleDynAnl.

The muscle recruitment problem specified by the members of this class can generally be formulated as

\[\begin{split}\begin{array}{ll} \text{minimize} & \text{``Primary Term''} + \text{``Auxiliary Terms''}\\ \\ \text{subject to} & \text{``Equilibrium equations''}\\ & \text{``Muscle forces cannot push''}\\ & \text{``Limits on exerted muscle forces''} \end{array}\end{split}\]

The primary specification is the type of the criterion. The table below provide an overview of the criterion types available, where the left column is possible values for the Type member variable. The auxiliary terms of the objective function, that can be either linear or quadratic, is not available for all criterion types. The availability is also indicated in the table.
