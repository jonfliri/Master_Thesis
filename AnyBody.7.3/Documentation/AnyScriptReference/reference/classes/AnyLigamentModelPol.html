AnyLigamentModelPol¶

Type

Creatable

Parent class

AnyLigamentModel

Description¶

This ligament model provides a constitutive force-length equation having a polynomial form. The parameters of the function are the slack length, the initial stiffness at this length, together with one point on the curve and the gradient at this point, i.e., the tangent stiffness. This provides an adjustable non-linear form of the constitutive (force-length) relationship. We shall refer to the values at this point as the nominal value: the nominal strain, force and stiffness.

The two stiffness values must be supplied as relative values referring to the nominal (mean) stiffness given by the two points. This nominal stiffness is F1/(eps1*L0) in N/m.

The polynomial function consists of constant, first, second, and fourth order terms.

It is possible to make the function continue as a linear function for deformations above the nominal strain. This can be used for defining linear constitutive relationships with a slack region.

Examples¶

Here is example of how to define a ligament model:

AnyLigamentModelPol MyLigament = {
    L0 = 1.30; // Slack length
    eps1 = 0.2; // Strain where F1 is valid
    F1 = 1000; // Force in the ligament at strain eps1
    a0 = 1; // Relative initial stiffness at L0
    a1 = 1; // Relative stiffness at F1
    LinRegionOnOff = On; // ensure lineraity above the point (eps1,F1)
};
