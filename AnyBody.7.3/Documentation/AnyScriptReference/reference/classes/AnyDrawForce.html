AnyDrawForce¶

Type

Creatable

Parent class

AnyDrawObject

Description¶

This drawing object class draws all forces that are derived from AnyForceBase

Examples¶

Here is an example to add a drawing object to a AnyForce3D class.

AnyForce3D F = {
  Flocal = {100, 0.0, 0.0};
  AnyRefFrame &Target = .Object;
  AnyDrawForce Drw_F = {
     Visible = On; // Switch on visibility in model view
  };
};
