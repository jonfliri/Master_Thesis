AnyDescription¶

Type

Creatable

Parent class

AnyProjectObject

Description¶

This class is used for defining descriptions for AnyProject related objects. When defined, the description is shown in the Project tabs right window pane. Only the user-defined portions of the AnyDescription are shown.

Using the members BodyHTML or HTMLURL, HTML can be used to format the contents. Apart from standard HTML formatting, special links to functions in the AnyBody GUI can be inserted. These links are made with standard HTML anchors like this

&lt;a href=”about:anybody ‘command argument’”&gt;LINK TEXT&lt;a/&gt;

For more details of the ‘command argument’ options, please refer to the Introduction Chapter, section “How to write AnyScript”, subsection “HTML-based descriptions”.

BodyHTML = "" ¶ The body text of the description in html format. The BodyHTML must be...

Class type: AnyStringVar
Eval moment: RuntimeVar
Default: ""

The body text of the description in html format. The BodyHTML must be html-formatted text and will be used inside the body-tag of a default AnyBody-formatted empty html-page. If BodyHTML is used it will take precedence over both the BodyText and HTMLURL.

HTMLURL = "" ¶ A URL link to a complete html page, which will be displayed as the...

Class type: AnyStringVar
Eval moment: RuntimeVar
Default: ""

A URL link to a complete html page, which will be displayed as the AnyDescription contents. The URL must point to a complete html text, and if included, it will take precedence over the normal BodyText, but not the BodyHTML.

The text of the tooltip shown when the mouse is over the object in the Project Tree. This functionality is not implemented yet.
