AnyMechOpr¶

Type

Abstract

Parent class

AnyMechOprBase

Child classes

AnyBodyLigamentLenCalOpr, AnyBodyMusFibAndTenLenCalOpr, AnyBodyMusTenLenCalOpr, AnyMechOprInvDynAnl, AnyMechOprKinAnl

Description¶

Base class for operations associated with the basic mechanical studies, i.e. AnyMechStudyBase classes. This base-class is derived from AnyMechOprBase and extended with pre- and post-process operations.

The PreOperation is executed before the body part of the AnyMechOpr itself, any PostOperation is executed after. Both operations are, by default, empty operations, which called be filled by the user e.g. with references to other operations (AnyOperation)
