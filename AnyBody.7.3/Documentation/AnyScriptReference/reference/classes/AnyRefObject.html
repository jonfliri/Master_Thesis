AnyRefObject¶

Type

Hidden

Description¶

Objects of this class are references to other existing objects in the model. Reference objects cannot be created directly by declaration of an AnyRefObject. Instead a reference is created by the class name of the referred class followed by the reference operator &, for instance

AnyObject& Ref2AnyObect = TheRealObject; // This is a reference to an AnyObject object.
