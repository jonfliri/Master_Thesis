AnyMetabModelSimple¶

Type

Creatable

Parent class

AnyMetabModelBase

Description¶

This class comprises a simple metabolic model that computes the metabolic power based on two efficiency coefficients, one for concentric and one for eccentric work.

The efficiency coefficients (\(\eta=\mathrm{Eta}\)) defines how much of the metabolic power (Pmet) is converted to actual mechanical power of the muscle (Pm). This implies that Pmet can be calculated as

\[P_{met} = \frac{P_{m}}{\eta}.\]

For the concentric case, this is less than 100%, which means that the metabolic power is larger than produced power. By default, the concentric efficiency ‘EtaCon’ is set to 25%, which implies that the metabolic power is 4 times larger than the actual produced mechanical power output of the muscle (i.e. when positive power is generated).

For the eccentric case, the mechanical power is negative (\(P_{m}<0.0\)), but this still cost positive metabolic power to maintain the force. Therefore, the efficiency coefficient must be negative to cancel the negative sign of \(P_{m}\). The default setting is that the negative (eccentric) mechanical power output of the muscle is 20% larger than the metabolic power required to produce and maintain it. This means the ‘EtaEcc’ is set to minus 120%.

This default values of EtaCon and EtaEcc follows the use in Voigt et. al., J.Biomech. 28, 281-291(1995). The default value of EtaCon=25% concentric work is for instance supported by findings in De Haan et.al., J.Exp.Biol. 141, 327-341(1989), while the default EtaEcc=-120% is derived from Asmussen, Acta Physiol.Scand. 28, 364-382(1953). In general, research in the field shows that these value must be assumed to be affected by many conditions, and therefore, it is a crude approximation to assume them to be constants as this metabolic model does. Thus, for more detailed studies, please consider one of the other metabolic models derived from class AnyMetabModelBase.

Examples¶

Here is simple code block to include the AnyMetabModelSimple in a study:

AnyMetabModelSimple SimpleMetabModel = {
    EtaCon = 0.25; // default value
    EtaEcc = -1.20; // default value
};

// Study in which to include the metabolic model
AnyBodyStudy Study = {
   Gravity = {0.0, -9.81, 0.0};
   MetabModel = &.SimpleMetabModel;
};
