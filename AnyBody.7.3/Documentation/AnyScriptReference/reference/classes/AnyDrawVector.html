AnyDrawVector¶

Type

Creatable

Parent class

AnyDrawObject

Description¶

A general vector drawing object. You can specify the 3D vector to be drawn by a mathematical expression, and similarly you can specify varying thickness, etc. The vector is attached to a reference frame (AnyRefFrame or derived class).

The reference frame is defined as either a member or the owner of the AnyDrawVector object. First priority is given to AnyRefFrame members and if none such exists the owner is taken. If neither are found it is an error.
