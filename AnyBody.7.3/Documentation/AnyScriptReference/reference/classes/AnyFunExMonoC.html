AnyFunExMonoC¶

Type

Creatable

Parent class

AnyFunExMonoBase

Description¶

Mono-function class for interfacing to externally coded functions in C++ implemented DLLs. This mono-function is to be used for user-defined AnyFunctions defined by means of the AnyFunEx class.
