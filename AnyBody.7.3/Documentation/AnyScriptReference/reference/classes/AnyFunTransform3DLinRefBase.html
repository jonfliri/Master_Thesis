AnyFunTransform3DLinRefBase¶

Type

Abstract

Parent class

AnyFunTransform3D

Child classes

AnyFunTransform3DLinRef, AnyFunTransform3DLinRef4Globals

Description¶

Base class for linear 3D transformations based on reference frames of mechanical objects (AnyMechObject and derived classes). It is typically AnyRefFrame or derived objects that are used here, but also other mechanical objects, such as surfaces, provide their frame of reference and can refered to in this function class.
