Type

Abstract

Parent class

AnyDrawObjectRigid

Child classes

AnyDrawCamera, AnyDrawNode, AnyDrawParamSurf, AnyDrawPointCloud, AnyDrawRefFrame, AnyDrawSTL, AnyDrawSurf

Description¶

This drawing object class is the base class for all geometrically rigid objects. This means that drawing object classes derived from this class define objects that do not change shape during motion of the mechanical elements of the model.

This extends its parent by facilities for controlling the appearance of the drawing, such as coloring and scaling.
