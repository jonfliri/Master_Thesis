AnyMetabModelB¶

Type

Creatable

Parent class

AnyMetabModelBase

Description¶

This class comprises the muscle-metabolism model presented in the paper Bhargava, L.J., Pandy, M.G. & Anderson, F.C., J.Biomech. 37, 81-88(2004).

The model by Bhargava et. al. determines the metabolic power of the muscle from the total external power and the rate of heat production (the heat rate), which implies that AnyBody can find the metabolic power if the heat rate is known. In the model by Bhargava et.al., the heat rate, \(\dot{H}\), is composed of four terms.

\[\dot{H} = \dot{A} + \dot{M} +\dot{S} + \dot{B}\]

respectively referred to as:

  • \(\dot{A}\): Activation power.

  • \(\dot{M}\): Maintenance power.

  • \(\dot{S}\): Shortening-lengthening power.

  • \(\dot{B}\): Basal power.

For further details, please refer to the paper and the AnyBody technical documentation on muscle modeling.

Related to the implementation and parameter/member naming in this class, please notice the following

  • ‘Con’ and ‘Ecc’ refer to concentric and eccentric contraction, respectively.

  • ‘Slow’ and ‘Fast’ refer to slow- and fast-twitch fibers, respectively.

  • The activation power uses the coeeificent phi0 assumed to be constant, where in the original paper it comprises of an additional term that decays towards zero with muscle activation time.

  • ‘AddMusclePowerForEcc’ enables you to add the external power produced by the muscle to the heat production in the eccentric cases. This is always done for the concentric case, but it is not obviously the case in the eccentric case from a mechanical point of view. Therefore, this switch. In the original paper by Bhargava this is indeed done and the parameters in the paper must be considered as calibrated under this assumption. Therefore, ‘AddMusclePowerForEcc’ is ‘On’ by default, even though theoretical argumentation suggests ‘Off’.

Examples¶

Here is simple code block to include the AnyMetabModelSimple in a study:

// Class template with default values shown
AnyMetabModelB BhargavaMetabolicModel = {
    //CoefBasalPower = 0.0225;
    //CoefSLPowerConIso = 0.16;
    //CoefSLPowerCon = 0.18;
    //CoefSLPowerEcc = 0.157;
    //CoefActPowerPhi0 = 0.06;
    //CoefActPowerSlow = 40.0;
    //CoefActPowerFast = 133.0;
    //CoefMaintPowerSlow = 74.0;
    //CoefMaintPowerFast = 111.0;
    //MuscleDensity = 1059.7;
    //AddMusclePowerForEcc = On;
    //AddMusclePowerForEccCoef = 1.0;
 };

// Study in which to include the metabolic model
AnyBodyStudy Study = {
   Gravity = {0.0, -9.81, 0.0};
   MetabModel = &.BhargavaMetabolicModel;
};
