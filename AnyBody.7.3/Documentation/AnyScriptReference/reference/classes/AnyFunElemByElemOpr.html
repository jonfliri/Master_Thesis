AnyFunElemByElemOpr¶

Type

Abstract

Parent class

AnyFunElemOpr

Child classes

AnyFunACos, AnyFunASin, AnyFunATan, AnyFunATan2, AnyFunAbs, AnyFunCeil, AnyFunCos, AnyFunCosh, AnyFunDiv, AnyFunExp, AnyFunFloor, AnyFunLog, AnyFunLog10, AnyFunMult, AnyFunPow, AnyFunRound, AnyFunSin, AnyFunSinh, AnyFunSqrt, AnyFunTan, AnyFunTanh

Description¶

Base class for a number of single argument functions that operates on each element independently. Therefore these functions can be applied to scalars, vector, matrices, and strutures of higher dimensionalities and they return values with the same dementionality as the argument.
