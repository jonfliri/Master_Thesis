AnyOutputBVH¶

Type

Creatable

Parent class

AnyOutputFile

Description¶

Class to save data from a BVH structured model in the BVH format or other text formats.

The class saves data from af BVH model defined by an AnyInputBVH object to a new file. By default, this new file is formatted as a new BVH file, but it can also be formatted any text format supported the parent class AnyOutputFile. In contrast to AnyOutputFile, this class automatically find the data to output from the BVH model. The pointer, BVHReader, must point to the AnyInputBVH object, which contains the BVH model.

Please notice that:

  • When using BVH format, the Hierarchy header section is copied one to one from the original header file, while the Motion section is written specifically using e.g. number format specification made in this object.

  • The BVH format assumes constant time step and a column with time values does not exist. If this object is requested to output data with varying time step size, it leads to an error. The data can, however, be written on other formats.

Examples¶

Here is simple code block of an AnyOutputBVH object that could be defined inside a study

AnyOutputBVH OutFileBVH = {
  BVHReader = &.BVHModel;
  FileName = "outputfile.bvh";
};

In the next example, the BVH format is switched off and thereby, the basic features of the parent class AnyOutputFile are enabled; here this is producing a text file.

AnyOutputBVH OutFileBVH_as_text = {
  BVHReader = &.BVHModel;
  FileName = "outputfile.csv";
  BVHFormatOnOff = Off;
};
