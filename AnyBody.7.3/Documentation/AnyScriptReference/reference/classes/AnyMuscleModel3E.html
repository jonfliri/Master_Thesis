AnyMuscleModel3E¶

Type

Creatable

Parent class

AnyMuscleModel

Description¶

This class contains a three element, Hill-type muscle model. The model comprises a contractile element (the active muscle fibers), a non-linear serial elastic element (the tendon) and a non-linear parallel elastic element (the passive part of the fibers). The contractile element is characterized by the force-length and the force-velocity relationship. The muscle fibers can have a pennation angle referred to as Gamma, i.e. Gamma0 for the nominal angle corresponding to the optimal fiber length.

The introduction of an pennation angle of the fibers imply that one must be aware of how various parameters are associated with the two reference systems. The following values refer to the pennate reference system:

  • nominal/optimal fiber length, ‘Lf0’

  • nominal muscle strength, ‘F0’

whereas the following values still refer to the muscle’s line of action (i.e. the reference system of kinematic measure of the muscle):

  • nominal tendon length and strain, ‘Lt0’ and ‘Epsilon0’;

  • quantities in the muscle object, which is using this muscle model, such as muscle length, ‘Lm’; all tendon related parameters; muscle strength, ‘Strength’; passive and active muscle forces, ‘Fp’ and ‘Fm’.

More details on muscle models and calibration can be found the tutorial about muscle modeling.

Input parameters, ‘Lfbar’, ‘Gammabar’ and ‘Epsilonbar’ are deprecated members, which were used before AnyBody version 7.1. Now they are replaced by similar values where ‘bar’ is replaced by ‘0’. New and deprecacted values have the same meaning.
