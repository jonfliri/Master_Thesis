AnyFunSquareWave¶

Type

Creatable

Parent class

AnyParamFun

Child classes

AnyFunSquareWaveThreshold

Description¶

Parameter function class making a smooth square step function, a square wave function. Smoothness is achieved by a smooth function interpolating the starting and the ending value of the steps over a period of time, i.e., the transition period given by dT. Transitions happens at specified times given by Ts.

The smoothness allows use of these functions in the specification of motion in kinematical drivers. A typical type of usage is weighting functions for instance for drivers relying on experimentally obtained data that are not valid over the entire time domain. With a AnyFunSquareWave-function one can make a weighting function with zero weight when the data are not valid, see also the derived class AnyFunSquareWaveThreshold.
