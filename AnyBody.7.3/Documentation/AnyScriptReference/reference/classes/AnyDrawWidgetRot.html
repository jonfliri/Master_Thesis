AnyDrawWidgetRot¶

Type

Creatable

Parent class

AnyDrawWidgetBase

Description¶

The AnyDrawWidgetRot class is used for creating graphical rotational widgets in the Model View.

 

Expected members¶

AnyStyleDrawMaterial (0..Inf) ¶ Graphical material specification, that overrides the color setting in RGB.

Class type: AnyStyleDrawMaterial
Number: 0..Inf

Graphical material specification, that overrides the color setting in RGB.

Required initialization members¶

RefFrame¶ An object-pointer to an Any3dObject, which is the object that the rotational...

Class type: AnyObjectPtrVar
Eval moment: Const

An object-pointer to an Any3dObject, which is the object that the rotational widget gets its refframe from.

Target¶ An object-pointer to an AnyFloat that is the target of the rotational...

Class type: AnyObjectPtrVar
Eval moment: Const

An object-pointer to an AnyFloat that is the target of the rotational widget. If the AnyFloat is an AnyMat33, the rotational widget will create a single rotational matrix as output in the Target. Otherwise, the Target AnyFloat will be written to as a number of independent rotational angles about the axes specified in the Axis1, Axis2 and Axis3 members.
