AnyInputXML¶

Type

Creatable

Parent class

AnyDataFileBase

Description¶

Class to read files in an XML format (see.

The class allows you to read the data in the file and it optionally can create a model with segments resembling the kinematical hierarchy in the file.

The data import can apply scaling parameters to the translation and rotations independently. This is useful for files using other units than meters and radians, since these units are used by the generated model. The scaling is only applied to that data in the Model section so the original data can be found in the Motion section.

Additionally, it is also possible to define filters for the translation as well as rotational data contained in the file. By default, one filter is included in the object, but additional filters can also be defined simply by constructing them inside the object or by creating references. Specification of which filter to use is made using ranslationalDataFilterIndex and RotationalDataFilterIndex for the translational and rotational data, respectively.

Special filter-options, Rotation2PiFixOnOff and Rotation1PiFixOnOff, for angular data allows to remove dicontinuities arising from angular values jumping any amount of full rotations or half rotations, repecitively. Angular data jumps of half rotations are related to the fact that Euler-like angles may describe the same 3D orientation with multiple data sets that typically differs by some integer times PI, i.e. half rotations, and a sign change for the 2nd of the three rotations. Notice that this type of discontinuity is related to a 180 degree switch of direction of 2nd axis of rotation. This angular discontinuity filters will first remove discontinuities of 2Pi (if swithced on) and subsequently, discontinuties of Pi (if switched on).
