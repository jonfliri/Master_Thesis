AnyFunTransform3DSTL¶

Type

Creatable

Parent class

AnyFunTransform3D

Description¶

3D transformation function based on two STL files using Radial Basis Functions (RBF).

Please see AnyFunTransform3DRBF for more detailed explanation of the RBF transform construction principles.

This class reads source and target triangulated surfaces from files defined by FileName0/FileName1 variables.

Note

Frequently surface models come from medical images, which typically operate in millimeters. Conversion between length unit systems can be achieved by specifying ScaleXYZ0/ScaleXYZ1 vectors.

The principles of transform construction: surfaces are assumed to be well aligned. If the surface registration has not been performed it can be done and specified in PreTransforms variable, where these transforms can be any of AMS-transforms, e.g. AnyFunTransform3DRBF, AnyFunTransform3DLin or AnyFunTransform3DLin2. Furthermore, a number of points (NumPoints) is seeded on the source surface and corresponding landmarks are found on the target surface by identifying closest vertices. These landmarks define an RBF transform in combination with RBF specific variables.

Examples¶

This particular example morphs a source thigh STL to a target thigh STL with the same topology. However if the STLs you are using do not have the same topology, adjust the following settings RBFDef.Type, RBFDef.Param, and UseClosestPointMatchingOnOff.

AnyFunTransform3DSTL STLTransform=
{
  PreTransforms = {&.AffineTransform}; // See AnyFunTransform3DLin2 Example
    RBFDef.Type = RBF_ThinPlate;       // works well for morphing long bones (femur/tibia) with same topology otherwise use RBF_Triharmonic and define RFBDef.Param
    AnyFixedRefFrame ref = {
      AnySurfSTL source = {
        FileName = Source_thigh;
        ScaleXYZ = {1, 1, 1};
        AnyFunTransform3D &pre = ...TSeg2ScaleFrame;
      };
      AnySurfSTL target = {
        FileName = Target_thigh;
        ScaleXYZ = {1, 1, 1};
      };
    };
    SurfaceObjects0 = {&ref.source};
    SurfaceObjects1 = {&ref.target};
    NumPoints = 5000;
    UseClosestPointMatchingOnOff = Off; // Default is 'On' for using closest-point matching; switch 'Off' if vertex-to-vertex matching is used.
    BoundingBox.ScaleXYZ = 5*{1, 1, 1};
    BoundingBox.DivisionFactorXYZ = 5*{1, 1, 1};
    BoundingBoxOnOff = On;
  };
};

Coefficients for the polynomial function. Each term of the polynomial must be given by three parameters, one for each direction in space, i.e., the PolynomCoefs are given as a (m x 3) matrix.
