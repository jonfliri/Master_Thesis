AnyFunTransform3D¶

Type

Creatable

Parent class

AnyFun3D

Child classes

AnyFunTransform3DIdentity, AnyFunTransform3DLin, AnyFunTransform3DLin2, AnyFunTransform3DLinRefBase, AnyFunTransform3DRBF, AnyFunTransform3DSTL

Description¶

Base class for 3D transformation functions. This type of function converts 3D (geometrical) vectors into new 3D vectors according to a given function built into the class. Such functions can be used for geometrical scaling.

This base class was originally an abstract class, but now it is creatable and contains the identity transform (interchangeable with AnyFunTransform3DIdentity).

Examples¶

We can use AnyFunTransform3D to reference an existing scaling function that is not defined inside particular file we are working in. For example the following AffineTranform requires that the sourcepoints are first transformed into the segment’s scale frame. So we can use AnyFunTransform3D to define &TSeg2ScaleFrame, using a reference to the existing scaling function.

AnyFunTransform3D &TSeg2ScaleFrame = Main.HumanModel.Scaling.GeometricalScaling.PelvisSeg.TSeg2ScaleFrame;

AnyFunTransform3DLin2 AffineTransform = {
  Points0 = .TSeg2ScaleFrame(.sourcepoints);
  Points1 = .targetpoints;
  Mode = VTK_LANDMARK_AFFINE;
};

This class can also be used to create a more complex scaling function made up of multiple transforms. For instance, this ScaleFunction will first apply an RBFTransform followed by a reversed affine transformation.

AnyFunTransform3D ScaleFunction = {
  PreTransforms = {&.RBFTransform, &.ReverseTransform};
};

Finally, you might come across the option to use AnyFunTranform3D within an AnyDrawSurf class. You can choose whether you’d like to insert an object reference (option A) or a full object definition (option B):

//option A
AnyFunTransform3D RegistrationTranform = {
  PreTransforms = {
    &Main.HumanModel.Scaling.GeometricalScaling.PelvisSeg.inv,
    &Main.HumanModel.BodyModel.Trunk.SegmentsLumbar.PelvisSeg.Scale_Trunk_Pelvis.ScaleAfterInterfaceMorphingDef.Scale.T1_Inv};
};

AnyDrawSurf target_pelvis= {
  FileName=TARGET_PELVIS;
  ScaleXYZ={1,1,1};
  RGB={0,1,0};
  AnyFunTransform3D &reg = .RegistrationTranform;
};

//option B
AnyDrawSurf target_pelvis= {
  FileName=TARGET_PELVIS;
  ScaleXYZ={1,1,1};
  RGB={0,1,0};
  AnyFunTransform3D RegistrationTranform = {
    PreTransforms = {
      &Main.HumanModel.Scaling.GeometricalScaling.PelvisSeg.inv,
      &Main.HumanModel.BodyModel.Trunk.SegmentsLumbar.PelvisSeg.Scale_Trunk_Pelvis.ScaleAfterInterfaceMorphingDef.Scale.T1_Inv};
  };
};
