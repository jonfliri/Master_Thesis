AnyInputFile¶

Type

Creatable

Parent class

AnyDataFileBase

Description¶

Class to read a space or comma separated values in an ASCII file. It reads multi-column data into the ‘Data’ structure. By default, the first column is treated as a “time” axis and read into the separate ‘T’ array. However, this separate handling of the the first column can be switched off by the ‘TOnOff’ switch.

The input-file object is able to reload the data file automatically during update of design data, for instance when an operation/analysis of a mechanical study starts. This auto-reloading must be turned on by the switch ‘ReloadOnOff’. In order to ensure that the object can operate, a least partially, when the data in the file is corrupted or non-existing default data can be supplied.

Notice the following about the data usage and reloading:

  • The static data allocation will be at least the size of the default data and the data in file at model loading.

  • Less data in the file will be extrapolated with constant values to fill the allocation size.

  • Excessive data during reloading of the data file will be truncated.

  • Data in the file has higher priority that default data; thus, default data is only used to set allocation size and when data in file is corrupted.
