AnyFunSquareWaveThreshold¶

Type

Creatable

Parent class

AnyFunSquareWave

Description¶

Parameter function class making a square wave function, see class AnyFunSquareWave, based on a data set and a threshold specifying the transitions.

This is particularly useful for making weighting functions based on sampled data, e.g. experimentally obtained data.

Transitions that are so closely spaced in time that the transition period cannot be contained are eliminated so that they do not enter the transitions at all.
