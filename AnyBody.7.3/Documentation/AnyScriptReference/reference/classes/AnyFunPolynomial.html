AnyFunPolynomial¶

Type

Creatable

Parent class

AnyParamFun

Description¶

Polynomial function.

The Polynomial function calculates each element of the output according to this expression:

\[f_i(t) ~=~ \sum_{j=0}^{n-1}\, P_{ij}\, t^{j}\]

where \(P_{ij}\) is the elements of the member PolyCoef, \(i\) is the index of the return value, and \(j\) is the counter over the elements in PolyCoef associated with the i’th return value.

The sizes and dimensionality of the return from this function is given by the member PolyCoef and therefore PolyCoef that must be initialized. PolyCoef is interpreted as an array of blocks with the same dimensionality and sizes as the return of the function, i.e., PolyCoef must have dimensionality one higher that the wanted return value.

In other word, if we indicate return values data block size like this \(\{XXX\}\) the member initialization must have the following form.

\[PolyCoef ~=~ \{~\{XXX\},~\{XXX\},~...~\};\]
