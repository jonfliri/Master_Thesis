AnyDrawGroup¶

Type

Creatable

Parent class

AnyObjectLayer

Description¶

This drawing group class is a class specially aimed at control of Model View actions, such as hiding, showing, setting transparency, etc. A drawing group object has the same options in relation to such Model View actions as other AnyFolder derived objects, but it does not have a particular rendering associated with the object. Instead it will transfer all Model View state actions performed on the object to all its members objects defined in the Objects pointer array.
