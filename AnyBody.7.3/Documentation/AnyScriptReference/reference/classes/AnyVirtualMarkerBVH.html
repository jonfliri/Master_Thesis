AnyVirtualMarkerBVH¶

Type

Creatable

Parent class

AnyOprData

Description¶

Class used to define virtual markers on a BVH model, see AnyInputBVH for more details.

Comment in the lines below when adding examples

Examples¶

Here is a virtual marker is defined in the Markers folder of an AnyInputBVH object. The marker is located on the root segment, called ‘Hips’, of the BVH model.

AnyInputBVH BVH_Model = {

  ... for more details see AnyInputBVH

  Markers = {
    AnyVirtualMarkerBVH HipsMarker = {
      RefFramePtr = &..Model.Hips.Seg;
    };
  };
};
