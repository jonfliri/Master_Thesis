AnyOutputGCD¶

Type

Creatable

Parent class

AnyOutputFile

Description¶

This output file class is specialized to prepare a data file on the GCD (Gait Cycle Data) format. This is specially aimed at making output for clinical gait analysis and possibly similar analysis of periodic motion. The format is a relatively simple ASCII format, where the desired data are cut into two intervals (one from left heel strike to left heel strike and the other from right heel strike to right heel strike) with a specified number of sampled points over the interval.

The gait cycles (for the left and right side) are defined separately using the time specifications LeftHeelStrike1 (first occurance of left heel stike) and LeftHeelStrike2 (second occurance of left heel stike) for the left side and RightHeelStrike1 and RightHeelStrike2 for the right side. These time specifications are in absolute time as used in the study. Points is used to specify the number of data points there should be written in the output file for each output variable. If Points does not match the analysis time of the study, the output data are linearly interpolated and re-sampled.

Links to output data can be done in two different ways: 1) Inside the AnyOutputGCD object, there are two pre-defined folders: Left and Right, and references to data made inside, e.g. the Left folder, will be written to the output file in the time interval specified for the left side, i.e. using LeftHeelStrike1 and LeftHeelStrike2. Similarly for the right hand side by placing data references inside the Right folder. 2) All data objects (AnyValue and derived) inside the object will also be written to the output file. Depending on their names, the varibles will be written in the left or right side gait cycle intervals: 1) If the variable does not contain the words “Left” or “Right”, it will be output to the file in both intervals, 2) if it contains the word “Left” it will be output in the left interval only, 3) if it contains the world “Right” it will be output in the right interval only, and 4) If it contains both “Left” and “Right”, the object will report an error as consistent naming of the variable in the file is not possible.

When writing the GCD file, the names of the AnyScript variables are changed: 1) “!Left” or “!Right” are prepended, depending on whether it is in the left or right side gait cycle intervals. 2) All dots in the full AnyScript name of the variable are removed to obtain more compact names.
