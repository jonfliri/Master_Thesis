Radial Basis Function (RBF) types.

A RBF is a function whose value depends only on the distance from the origin, i.e., \(\phi = f(r,\beta)\) where r is usually the Euclidean distance and \(\beta\) is some function parameter.

  • Biharmonic basis function: \(\phi~=~r\)

  • Triharmonic basis function: \(\phi~=~r^3\)

  • Thin-plate basis function: \(\phi~=~r^2~log(r)\)

  • Gaussian basis function: \(\phi~=~exp({-\beta~r^2})\)

  • Multi-Quadratic basis function: \(\phi~=~\sqrt{(r~-~\beta)^2}\)

  • Multi-Quadratic basis function: \(\phi~=~\frac{1}{\sqrt{(r~-~\beta)^2}}\)
