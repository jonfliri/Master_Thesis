AnyProjectTaskOperation¶

Type

Creatable

Parent class

AnyProjectTaskBase

Description¶

This class is a Task-class specialization dedicated to executing a particular operation (AnyOperation object) of the model. In addition to executing a particular task, a particular Model View Definition can be associated with this task.
