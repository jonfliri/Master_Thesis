AnyForce3D¶

Type

Creatable

Parent class

AnyAppliedForce

Description¶

A three-dimensional concentrated force (a vector). The force can be specified in both global and local coordinates but not simulataneously.

Examples¶

Here is simple example of how to add a force working in local coordinates of the target object.

AnyForce3D AppliedForce = {
  Flocal = {100, 0.0, 0.0};
  AnyRefFrame &obj = TargetObject;
};

Here is simple example of how to add a force working in global coordinates.

AnyForce3D AppliedForce = {
  F = {100, 0.0, 0.0};
  AnyRefFrame &obj = TargetObject;
};
