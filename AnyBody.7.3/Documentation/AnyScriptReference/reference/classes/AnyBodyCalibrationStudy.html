AnyBodyCalibrationStudy¶

Type

Creatable

Parent class

AnyKinStudy

Description¶

The AnyBodyCalibrationStudy contains functionality for adjusting the musculoskeletal models systematically. We refer to this as a calibration of the model, therefrom the origin of class name.

These models often have many parameters, such that making manual adjustment is exhaustive or impossible. Moreover adjustment of parameters for given parts of the model, such as muscles or ligaments, are typically based on some global measures of the whole body or the particular limb. For instance, one may know fairly accurately in which position of the knee, the thigh muscles have the optimal position, but converting this into an accurate adjustment of the muscle and tendon length requires detailed information about the muscle kinematics.

This study provides general and easy applicable parameter adjustment schemes of this sort. You supply a kinematically determinate model to the study (i.e. drivers of all degrees of freedom), similarly to when using the AnyBodyStudy for kinematic or inverse dynamic analyses. The definition of the kinematics, as either a fixed position or a motion, defines the positions for which the model’s parameters are adjusted.

The model needs only include the elements that are needed to define the elements to be calibrated. Thus only the muscles and ligaments to be calibrated by this study should be included, along with the mechanical parts they are attached to, i.e., segments, joints, and drivers.

The following types of calibrations are available:

Muscle Fiber and Tendon Length Calibration

In this calibration method, the muscle’s fiber and tendon lengths are adjusted according to the given range of motion. The range of motion is given by the motion applied in the study; thus it is important to provide the calibration study with an actual motion, not just a single posture. The fiber and tendon length are adjusted so that the actual fiber lengths are specified ratios of the optimal fiber length at the extreme values of muscle tendon unit for the given the range of motion. The specified ratios are given by the two members RminMuscleFiber and RmaxMuscleFiber. This method can optionally assume full or zero tensional force, i.e., in the former case the maximal strain tendon specified for the muscle is included.

Muscle Tendon Length Calibration

In this calibration method, the length of the muscle tendons are adjusted so that the muscle’s fiber length coincide with the optimal fiber length for the given position of the system. This method assumes full tensional force, i.e., it includes the maximal strain tendon specified for the muscle.

Ligament Length Calibration

In this calibration method, the length of the ligaments are adjusted so that the given position of the system corresponds to the undeformed ligament, i.e., the zero force position.

Notice that if motion has be specified for the system, then the average of the bounds of the range of motion for each muscle or ligament is used as the “given position”.

Notice also that not all types of muscle models (AnyMuscleModel) and ligament models (AnyLigamentModel) necessarily are handled by these calibrations.

Examples¶

Here is a simple calibration study to calibrate the Tensor Faciae Latae muscle:

AnyBodyCalibrationStudy TensorFasciaeLataeCalibration = {

  AnyFolder &Model = Main.MyModel; // Include the desired model
  AnyFolder &CalibrationDrivers = Main.MyCalibrationDrivers; // Include the calibration position drivers

  nStep = 2; // should be equal to the number of calibration positions

  // Array of muscles to be calibrated
  MuscleArr = {
      &Main.HumanModel.BodyModel.Right.Leg.Mus.TensorFasciaeLatae1,
      &Main.HumanModel.BodyModel.Right.Leg.Mus.TensorFasciaeLatae2
  };

  RminMuscleFiber = {0.5,0.5}; // Lower fiber length ratios

  RmaxMuscleFiber = {1.2, 1.2}; // Upper fiber length ratios
};
