AnyFunTransform3DLinRef2¶

Type

Creatable

Parent class

AnyFunTransform3DLinRef

Description¶

Linear 3D transformation based on reference frames given by mechanical objects (AnyMechObject and derived classes).

This class is basically identical to AnyFunTransform3DLinRef except that it can transform to global coordinates, and not only to the root-frame like AnyFunTransform3DLinRef. Therefore, it does not have the restriction that Ref and RefTarget must have the same root-reference. This implies that is can also transform between references systems that are moving with respect to each other.

This functionality is used in the two member-(mono)functions that allow to override the members Ref and RefTarget in expressions.

The limitation of this function, compared to AnyFunTransform3DLinRef and most other AnyFunTransform3D, is that the return values will be ready later (at the Evaluation Moment called PosVar) and thus cannot be used for typical scaling of model design (design scaling is evaluated at the earlier Evaluation Moment called DesignVar).
