AnyKinMeasureProd¶

Type

Creatable

Parent class

AnyKinMeasureCombBase

Description¶

This kinematic measure class is deprecated. It was introduced as part of AnyBody v.7.2 beta program, but discontionued before a final release of AnyBody v.7.2. Its purpose was to introduce time-varying coefficients in products with kinematic measures.

It is strongly recommended not to use this class as it will be removed in future versions.

Please use AnyKinMeasureLinComb, which was improved for AnyBody v.7.2, so it offers the same functionality as AnyKinMeasureProd. Please also see AnyKinMeasureQuadComb that is an extension of AnyKinMeasureLinComb offering quadratic combination of kinematic measures.

RefFrameOutput.Jacobians¶ Data structure for holding Jacobian elements of the measure with respect to...

Class type: AnyFloat
Eval moment: VelVar
Default: "Dependent"

Data structure for holding Jacobian elements of the measure with respect to the degrees of freedom of the reference frames in ‘DepRefFrameArr’. The structure is 3-domensional. First dimension contains on item for each frame in ‘DepRefFrameArr’, second dimension one for each dimension of the measure, and finally, in the third dimension, 6 jacobian elements (3 translations and 3 rotations refering to global axes).
