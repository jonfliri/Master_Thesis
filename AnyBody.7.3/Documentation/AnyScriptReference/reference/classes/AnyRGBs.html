AnyRGBs¶

Type

Creatable

Parent class

AnyFloat

Child classes

AnyRGB

Description¶

Value type holding multiple RGB (color) values such as AnyRGB. Can be used to create arrays and matrices of RGB values. Each of the RGB values is basically a three component vector, and can only hold values in the range from zero to one.

Assignment with values outside this range will be corrected during expression validation steps and accompanied by a warning.

Examples¶

Here is simple code block to show a simple use case of nesting three RGB values in a AnyRGBs class:

AnyRGBs RGBarray = { {1,0,0}, {0,1,0}, {0,0,1} };

You could also create a matrix of RGB values if needed:

// Single RGB value using the AnyRGB class
AnyRGB red = {1.0, 0.0, 0.0};
AnyRGB green = {0.0, 1.0, 0.0};
AnyRGB blue = {0.0, 0.0, 1.0};

// Matrix of RGB values
AnyRGBs RGBarray = {red, green, blue};

// Tensor of RGB values
AnyRGBs RGBmatrix = { RGBarray, {red, green, blue} };
