AnyDrawSurfDynShape¶

Type

Creatable

Parent class

AnyDrawSurf

Description¶

This class draws a 3-D geometry that may change shape dynamically, i.e. over time. It is derived from AnyDrawSurf, which draws rigid geometries, and therefore it inherits most of its properties from this class. The main difference is that the AnyDrawSurfDynShape can hold a dynamical (time-dependent) 3D transformation function (i.e. special classes derived from class AnyFunTransform3D). This also implies that the update of shape for this type of drawing objects happens later and more often for the AnyDrawSurfDynShape than for AnyDrawSurf. Therefore, It should be used only when needed. For any rigid surface use AnyDrawSurf.

The late and frequent shape update also implies that surface infomation retrieved from member-functions, such as GetVertices(), will be available late. Specifically, it will not be available during design update (i.e. the DesignVar evaluation moment) and therefore, AnyDrawSurfDynShape objects’ vertices cannot be used as part of design parameterization. Only rigid surfaces (i.e. AnyDrawSurf) can be part of the design parameterization.

For more details, please refer to AnyDrawSurf.
