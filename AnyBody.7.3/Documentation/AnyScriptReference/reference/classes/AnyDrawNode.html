AnyDrawNode¶

Type

Creatable

Parent class

AnyDrawObjectRigidFormatted

Child classes

AnyDrawNodes

Description¶

This object draws a single node (AnyRefFrame) if it is put inside it.

Examples¶

Here is a minimal working example, using the default settings of the class.

AnyRefNode Node = {
  sRel = {0.1, 0, 0};
  AnyDrawNode drw = {};
};

Here is a another example showing how to control the size, color and opacity.

 AnyRefNode Node = {
   sRel = {0.1, 0, 0};
   AnyDrawNode drw = {
      ScaleXYZ = {0.05,0.05,0.05};
      Opacity = 0.8;
      RGB = {1,0,0};
  };
 };
