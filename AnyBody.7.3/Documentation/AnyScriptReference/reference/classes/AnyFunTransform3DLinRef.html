AnyFunTransform3DLinRef¶

Type

Creatable

Parent class

AnyFunTransform3DLinRefBase

Child classes

AnyFunTransform3DLinRef2

Description¶

Linear 3D transformation based on reference frames given by mechanical objects (AnyMechObject and derived classes). It is typically AnyRefFrame or derived objects that are used here, but also other mechanical objects, such as surfaces, provide their frame of reference and can refered to in this function class.

This function transforms the argument values from the reference frame given by the member Ref to the target frame given by the member RefTarget. Intermediately, it used the so-called root frame of the Ref-object. The root frame is the for instance the body-fixed frame of a segment (AnySeg) or a fixed frame (AnyFixedRefFrame) to which the Ref-object is attached. This implies that the function can only do transformations between reference frames attached rigidly to the same root frame.

This limitation allow the function to return results that are not depending on the positions of the mechanical sysem and thus it can be calculated during design updates, which is the typical Evaluation Moment of model scaling.

This limitation also imply that this function cannot calculate global coordinates, unless Ref is a fixed frame aligned with the global system.
