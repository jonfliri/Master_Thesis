AnyOutputFolder¶

Type

Abstract

Parent class

AnyOprBase

Description¶

Objects of this class store the derived/computed output data from the analyses that are carried out in the studies (i.e. AnyStudy and derived classes). It also contains an AnyAbscissaFolder object that stores the input or independent parameters of a given analysis.

The actual values of output for each set of input parameter values are stored in objects of class AnyOutputFloat that refer to the actual AnyValue objects and save their current value at designated points in the actual analysis. For more details on these designated points please refer to the description of the actual study in question.

An AnyOutputFolder object may contain the AnyOutputFloat objects in a folder that mimic the structure of the real value objects in the study.
