max()¶

Implemented by

AnyFunVecMax

Description¶

The maximum of all elements in a vector.

The function applies to matrices and strutures of higher dimensionalities as well. In such cases, the function processes all vectors of the first dimension and returns a similar structure without this dimension.

Call signature¶

max(Arg:AnyFloat) -> AnyFloat
max(Arg:AnyInt) -> AnyInt
