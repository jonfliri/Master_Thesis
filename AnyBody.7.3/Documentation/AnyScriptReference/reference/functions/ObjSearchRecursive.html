ObjSearchRecursive()¶

Implemented by

AnyFunObjSearchRecursive

Description¶

Recursive search function for finding objects in a branch of the model.

The function returns an array of object pointers (class AnyObjectPtr) matching the search criterion. The function searches recursively down in the Model Tree from the root object given as first argument. The function searches by name (as second argument) with wildcard characters (,?) allowed. The ‘’ character indicates “any substring” in the name token, whereas the ‘?’ character indicates “any single character”. In addition, it may take a class name as a third a argument and a integer limitting how deep in the Model Tree it will search, i.e. how many levels down from the root object.

Call signature¶

// ObjSearchRecursive() with 2 arguments
ObjSearchRecursive(RootName:AnyStringVar, ObjectSearchString:AnyStringVar) -> AnyObjectPtr

// ObjSearchRecursive() with 3 arguments
ObjSearchRecursive(RootName:AnyStringVar, ObjectSearchString:AnyStringVar, ClassName:AnyStringVar) -> AnyObjectPtr

// ObjSearchRecursive() with 4 arguments
ObjSearchRecursive(RootName:AnyStringVar, ObjectSearchString:AnyStringVar, ClassName:AnyStringVar, LevelsDown:AnyIntVar) -> AnyObjectPtr
