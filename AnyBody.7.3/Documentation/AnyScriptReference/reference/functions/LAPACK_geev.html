LAPACK_geev()¶

Implemented by

AnyFunGenEigenProblem

Description¶

This function solves an eigenproblem and returns computed eigen values/vectors on request. It is a wrapper for geev function of LAPACK library.

  • \(A\) - Input matrix, \(n\times n\).

  • \(jobvr\) - “V” specifies to output right eigen vectors, which provide eigen vectors in the formulation: \(A\times v= \lambda\times v\), “N” - suppresses this output.

  • \(jobvl\) - “V” specifies to output left eigen vectors, which provide eigen vectors in the formulation: \(v^{T}\times A=v^{T}\times\lambda\), “N” - suppresses this output.

  • \(eval\) - “V” specifies to output eigen values, both real and imaginary parts, “N” - suppresses this output.

Please note that based on the requested parameters the output is constructed in the following order: right eigen vector matrix (\(n\times n\)), left eigen vector matrix (\(n\times n\)), real part of eigen values (\(n\times 1\)), and imaginary part of eigen values (\(n\times 1\)). Where \(n\) is the matrix dimension along one of the axes.

Call signature¶

// LAPACK_geev() with 4 arguments
LAPACK_geev(Amat:AnyMatrix, jobvr:AnyString, jobvl:AnyString, eval:AnyString) -> AnyFloat
